PyDay Angola

O primeiro passo para a PyCon Angola

O que é o PyDay Angola?

O PyDay Angola é um evento preparatório para a PyCon Angola, uma iniciativa para aquecer a comunidade Python no país. Será realizado no Namibe, uma região com grande potencial tecnológico e que acolherá o primeiro passo da comunidade Python em Angola.

Building the Python Community in Angola — este é o nosso propósito. O PyDay é o ponto de partida para conectar entusiastas, desenvolvedores, estudantes e empresas que querem fazer parte do ecossistema Python em Angola.
